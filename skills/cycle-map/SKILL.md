# 산업 사이클 맵 — 다수 산업 위치 표

프프서울 1주차 첫 화면 워크샵용. 한 산업을 깊게 들어가기 전, 전체 그림을 한 페이지 표로 잡는 단계입니다.

## 코호트 원칙

- 종목 추천 금지. 산업/섹터 레벨에서 멈춤.
- 모든 추정은 가설. 출처 함께.
- 결론보다 **비교 절차** 자체를 학습.

## 절차

### 1) 산업 리스트 결정

사용자에게 묻습니다:

> 어떤 산업들을 비교할까요? 콤마로 구분해 입력하거나, 비워두면 디폴트 10개로 진행합니다.

**디폴트 10개**:
반도체, 2차전지, 자동차, 조선·방산, 헬스케어 바이오, 금융, 부동산·건설, 에너지·정유, 미디어·광고, 항공·여행

### 2) 각 산업의 사이클 위치 추정

각 산업에 대해 **간단하게** (산업당 2-3문장):

- 현재 단계 (초기/중기/후기 확장, 수축)
- 직전 6-12개월 모멘텀 방향 (개선/유지/약화)
- 핵심 1개 신호 (수요·재고·capex·마진·자본조달 중 결정적인 것)
- 출처 1-2개 (실시간 데이터가 어렵다면 "최근 분기 추세" 정도로 명시)

### 3) 표 출력

다음 형식 마크다운 표로:

```markdown
| 산업 | 사이클 단계 | 모멘텀 | 핵심 신호 | 투자 매력도 (1-5) | 비고 |
|---|---|---|---|---|---|
| 반도체 | 중기 확장 | 개선 | 메모리 가격 회복 | 4 | HBM 수요 주도 |
| ... | ... | ... | ... | ... | ... |
```

**투자 매력도(1-5) 산정 기준**:
- 5: 초기 확장 + 강한 모멘텀
- 4: 초기/중기 확장 + 유지 또는 개선
- 3: 중기 확장 + 유지
- 2: 후기 확장 또는 모멘텀 약화
- 1: 수축 또는 강한 부정 신호

### 4) 딥다이브 추천 1개

표 아래에:

```markdown
## 1주차 딥다이브 추천
**[산업명]** — 이유: ...

`/pp-seoul-cohort:industry-cycle` 로 깊게 분석해 보세요.
```

선정 기준: 매력도 높음 + 정보 접근성 좋음 (한국/미국 양쪽에서 데이터 풍부).

### 5) 반박 가능성

표 아래에:

```markdown
## 표 전체의 반박 가능성 2개
1. 매크로 환경이 [예: 금리 동향]에 따라 사이클이 동시에 흔들릴 수 있음.
2. 사이클 단계 판정에 6-12개월 후행 지표가 섞여 있어 실시간 판단과 시차 발생 가능.
```

## 안전 규칙

- 종목/티커 출력 금지. 산업명만.
- 매력도 점수는 절대값이 아니라 **상대 순위**로 해석되도록 비고에 명시.
- 모르는 산업은 "데이터 부족, 추가 리서치 필요"로 솔직히 표기.
